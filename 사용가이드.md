# MusicDown 플랫폼 사용 가이드

## 🚀 빠른 시작

### 1단계: Superuser (관리자) 계정 생성
**반드시 먼저 실행하세요!**

1. `create_superuser.bat` 파일을 더블 클릭
2. 다음 정보 입력:
   - Username: admin (또는 원하는 이름)
   - Email: admin@example.com
   - Password: (안전한 비밀번호 입력, 최소 8자)
   - Password (again): (동일한 비밀번호 재입력)

### 2단계: 서버 실행

1. `run_backend.bat` 더블 클릭 (백엔드 서버 시작)
2. `run_frontend.bat` 더블 클릭 (프론트엔드 서버 시작)

### 3단계: 로그인

1. 브라우저에서 `http://localhost:5173` 접속
2. 우측 상단 "Login" 버튼 클릭
3. 1단계에서 만든 Superuser 계정으로 로그인

### 4단계: 음악 업로드

1. 로그인 후 상단 네비게이션 바에 **노란색 "Upload" 버튼**이 보입니다
2. Upload 버튼 클릭
3. 다음 정보 입력:
   - Track Title: 곡 제목
   - Duration: 곡 길이 (초 단위, 예: 180)
   - Artist Name: 아티스트 이름
   - Album Title: 앨범 제목
   - Track File: MP3 파일 선택 (필수)
   - Preview File: 미리듣기 파일 (선택)
   - Album Cover: 앨범 커버 이미지 (선택)
4. "Upload Track" 버튼 클릭

### 5단계: 음악 다운로드

1. 홈페이지 또는 검색 페이지에서 곡 선택
2. 곡 상세 페이지로 이동
3. **"Download" 버튼** 클릭
4. 파일 저장 위치 선택 후 저장

## ⚠️ 문제 해결

### Upload 버튼이 안 보여요
- **원인**: 일반 사용자 계정으로 로그인했거나, Superuser 계정을 만들지 않음
- **해결**: 
  1. 로그아웃
  2. `create_superuser.bat` 실행하여 관리자 계정 생성
  3. 관리자 계정으로 다시 로그인

### 다운로드가 안 돼요
- **원인**: 로그인하지 않았거나, 파일이 서버에 없음
- **해결**:
  1. 로그인 확인
  2. 백엔드 서버가 실행 중인지 확인 (`run_backend.bat`)
  3. 업로드한 파일이 `backend/media/` 폴더에 있는지 확인

### 서버가 시작 안 돼요
- **백엔드**: 
  - `backend` 폴더에서 터미널 열기
  - `..\venv\Scripts\activate` 실행
  - `python manage.py migrate` 실행
  - `python manage.py runserver` 실행
  
- **프론트엔드**:
  - `frontend` 폴더에서 터미널 열기
  - `npm install` 실행
  - `npm run dev` 실행

## 📝 주요 기능

1. **음악 검색**: 상단 "Search" 메뉴에서 곡/아티스트/앨범 검색
2. **Hover-to-Play**: 검색 결과에서 곡 위에 마우스를 올리면 자동 재생
3. **글로벌 플레이어**: 하단에 고정된 플레이어로 페이지 이동 중에도 재생 유지
4. **다운로드 기록**: "My Music" 메뉴에서 다운로드한 곡 확인

## 🔧 기술 스택

- **Backend**: Django + Django REST Framework
- **Frontend**: React + TypeScript + Vite
- **Database**: SQLite (개발용)
- **Authentication**: Token-based (Django REST Framework Token)
